# Speckit 文档

Speckit 项目文档 - Spec-Driven Development (SDD) 规范驱动开发完整指南

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run docs:dev
```

访问 http://localhost:8080 查看文档

### 构建文档

```bash
npm run docs:build
```

构建后的文件在 `docs/.vuepress/dist` 目录

## 文档结构

- **介绍** - 了解 SDD 和 Speckit 的基础知识
- **最佳实践** - 学习如何编写高质量的规范
- **使用指南** - 实际应用和工具集成

## 关于 Speckit

Speckit 是 GitHub 提供的一个工具包，旨在帮助开发团队采用**规范驱动开发（Spec-Driven Development，SDD）**的方法。SDD 强调在开发过程中首先定义清晰的规范，然后根据这些规范进行实现，以确保软件的质量和一致性。

了解更多：[GitHub Spec-Kit](https://github.com/github/spec-kit)
