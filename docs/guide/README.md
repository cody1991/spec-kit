## 概述

Speckit 是 GitHub 开发的工具包，用于支持规范驱动开发（Spec-Driven Development，SDD）方法论。SDD 是一种软件开发方法，强调在编写代码之前先定义清晰的规范。

## 核心价值

1. **提高代码质量** - 通过先定义规范，确保实现符合预期
2. **改善团队协作** - 规范作为团队沟通的共同语言
3. **减少返工** - 提前发现和解决问题
4. **文档即代码** - 规范本身就是最好的文档

## 适用场景

SDD 特别适合以下场景：

- API 设计和开发
- 微服务架构
- 前后端协作
- 跨团队项目
- 需要严格接口定义的项目
